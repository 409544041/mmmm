Mini Mike's Metro Minis

A city-themed collection of over 400 cute models you can remix or use outright in your own works

Download this project as a .zip file Download this project as a tar.gz file

Screenshots

Formats included

/vox is the original .vox files, created in MagicaVoxel. This is compatible with all sorts of voxel software like VoxelShop, Qubicle 2, etc.

/collada is optimized polygon files for use in 3d software packages like Cinema4d, 3D Studio, Maya, or conversion to other formats like FBX for use in game frameworks.

/unity-plugin contains all the models already setup for Unity3D with box colliders, wheel colliders on the vehicles and so on. Getting 400+ models ready for use in your game can be a chore, look how nice it is to have it all done for you!

Parts

  • 91 characters

  • 35 residential buildings

  • ...and another 200+ props and models for filling out the city
